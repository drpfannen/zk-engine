# ZK Engine - installable PWA

Static, offline-capable PWA. Upload every file in this folder to the **root** of a
GitHub repo, then enable **Settings -> Pages -> Deploy from a branch -> main / root**.
Open the resulting https URL on your phone and use Add to Home Screen.

Files: index.html, sw.js, manifest.webmanifest, icon-192.png, icon-512.png,
icon-512-maskable.png, apple-touch-icon.png
